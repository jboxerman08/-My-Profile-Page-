RentAMon
Your monument shipped anywhere in the world!

The goal of this project was to collaborate with other developers/classmates to recreate an AirBNB clone. The project combined user authentification and authorization, the implementation of a functional map with location markers and search bar to find a monument for rent, with a basic booking system that handles requests between a user and monument owner.

Clean Kut's
Your Barber Around the Corner

The goal of the project was to create a fully functional app from start to finish as the final group project of the Le Wagon bootcamp. As a team, we created a mobile friendly app with different user profiles, displaying nearby barbers on a map with barber location markers. The client can view barber's schedule and book an appointment through a booking system that handles requests between a client and barber. Another feature that was implemented was a chat system where the barber and client can contact each other via direct messages.
