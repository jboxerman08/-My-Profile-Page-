School - Lewagon Coding Bootcamp Projects

Having spent over 7 years in the workforce as an experienced assembler and quality assurance inspector, I made the career switch towards web development, graduating from the Le Wagon Coding Bootcamp in November 2011 as a full-stack developer.

Driven by curiosity and a passion for technology, I decided to dive into coding and explore the endless possibilities coding offers. This change has allowed me to start working at Diff Agency as of November 2011, joining the front-end development team.

Throughout the Le Wagon journey, we learned how to code several projects using a diversified stack, with a focus on Ruby/Ruby on Rails:

Watch List

The goal of this challenge was an introduction to creating basic user movie lists/bookmarks. With the implementation of cloudinary, the user has the option of uploading a personalized background for their user-created list, choosing movies from a small selection of movies from the Movie DB through an API.

Visit Watch List!

RentAMon

The goal of this project was to collaborate with other developers to create an AirBNB clone. The project combines user authentification and authorization, a functional map with location markers and search bar to find a monument for rent, with a basic booking system that handles requests between a user and monument owner.

Visit RentAMon!

Clean Kut's

The goal of the project was to create a fully functional app from start to finish as the final group project of the Le Wagon bootcamp. As a team, we created a mobile friendly app with different user profiles, displaying nearby barbers on a map with barber location markers. The client can view barber's schedule and book an appointment through a booking system that handles requests between a client and barber. Another feature that was implemented was a chat system where the barber and client can contact each other via direct messages.

Visit Clean Kut's!
