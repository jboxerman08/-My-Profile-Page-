Bootcamp Projects

Watch List: Create now, watch later!

The goal of this challenge was an introduction to creating basic user movie lists/bookmarks. With the implementation of cloudinary, the user has the option of uploading a personalized background for their user-created list, choosing movies from a small selection of movies from the Movie DB through an API.

RentAMon: Your monument shipped anywhere in the world!

The goal of this project was to collaborate with other developers/classmates to recreate an AirBNB clone. The project combined user authentification and authorization, the implementation of a functional map with location markers and search bar to find a monument for rent, with a basic booking system that handles requests between a user and monument owner.
